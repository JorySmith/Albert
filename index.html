Getting Started

Things to be aware of when using this problem solving application
  • Trigger > Reaction (how this application is formatted)
  • Focus on triggers, what we see in the world that reminds us to take a certain high-quality action
  • Any Trigger > Write about it > Dramatically reduces stress, improves analysis
  • Any Trigger > It's probably been done before, search, get exposed to new ideas
  • Solution/Conclusion > Why? Reasoning and Evidence to support this statement
  • "If I had an hour to solve a problem, I'd spend 55 minutes thinking about the problem and 5 minutes thinking about solutions." - Albert Einstein
  • The first draft of anything is shit. - Hemingway
The Problem Solving Process

Not sure where to start? Emotional? Write about the Situation
  • Situation > Write about it, gather thoughts and info, What's wrong? What's going right? How do you feel?
  • Situation > Identify Problems > Go to Problems

Audience decides Values
  • Audience > Write about it, decides everything, brainstorm, develop it fully, correctly ID the audience, fully understand it
  • Audience > Why? 5 Why's to Deeper Audiences
  • Audience > Values

Values reveal Goals/Tasks, Audience decides Values
  • Values > Write about it, brainstorm, develop it fully, correctly ID the value, fully understand it
  • Values > Why? 5 Why's to Deeper Values
  • Values > Also a Solution to what Problem? > Redo Problems for it
  • Values > To what Audience? > Go to Audience
  • Values > Refine, reduce data, prioritize
  • Values > Create a To-Do List, Plan, Schedule, Deadline, move on and relax
  • Values > Goals/Tasks

Goals/Tasks reveal Problems, Values reveal Goals/Tasks
  • Goals/Tasks > Write about it, brainstorm, develop it fully, correctly ID the goal, fully understand it
  • Goals/Tasks > Why? 5 Why's to Deeper Goals
  • Goals/Tasks > Also a Problem > Go to Problems
  • Goals/Tasks > Goal/Task to what Value?
  • Goals/Tasks > Break into Smaller Ones, make incremental progress, scale up, chip away at it
  • Goals/Tasks > Identify Future Ones to solve and prepare for now
  • Goals/Tasks > Refine, reduce data, prioritize
  • Goals/Tasks > Create a To-Do List, Plan, Schedule, Deadline, move on and relax
  • Goals/Tasks > Problems

Problems reveal Solutions, Goals/Task reveal Problems
  • Problems/Pains/Fears/Dangers > Write about it, obstacles/restrictions, brainstorm, develop it fully, correctly ID the problem (are there deeper problems?), fully understand it
  • Problems/Pains/Fears/Dangers > Why? 5 Why's to Deeper Problems > Redo Problems for each
  • Problems/Pains/Fears/Dangers > Break into Smaller Ones, make incremental progress, scale up, chip away at it
  • Problems/Pains/Fears/Dangers > Identify Future One to solve and prepare for now
  • Problems/Pains/Fears/Dangers > Also a Solution to what Problem? > Redo Problems for it
  • Problems/Pains/Fears/Dangers > Problem to what Goal? > Go to Goals
  • Problems/Pains/Fears/Dangers > Refine, reduce data, prioritize
  • Problems/Pains/Fears/Dangers > Create a To-Do List, Plan, Schedule, Deadline, move on and relax
  • Problems/Pains/Fears/Dangers > Solution

Solutions need Tested, Problems reveal Solutions
  • Solutions > Write about it, brainstorm, pros and cons, cost-benefit analysis, develop it fully, correctly ID the solution, fully understand it
  • Solutions > Why? 5 Why's to Deeper Problems
  • Solutions > Come with Problems, Consequences? Cons? > Go to Problems
  • Solutions > To what clearly wrong Solution that causes outrage? Cunningham's law
  • Solutions > Solution to what Problem/Goal/Task? > Go to There
  • Solutions > Refine, reduce data, prioritize
  • Solutions > Create a To-Do List, Plan, Schedule, Deadline, move on and relax
  • Solutions > Need Tested

Tests need Analyzed, Solutions need Tested
  • Tests > Get real-world feedback, ground truth, evolution by natural selection
  • Tests > Break into Smaller Ones, make incremental progress, chip away at it
  • Tests > Results Analyzed

Analyzed Results need Applied, Tests need Analyzed
  • Analyze Results > What went well, what didn't go well, what you problems did you avoid, what goals did you achieve
  • Analyze Results > What gets measured, gets improved
  • Analyze Results > Make incremental progress, chip away at it
  • Analyze Results > Results Applied

Applied Analysis reveals what to Practice, Analyzed Results need Applied
  • Apply Analysis > Adapt, improve, make changes, iterate, progress over perfection

Practice improves performance, Applied Analysis reveals what to Practice
  • Practice > Spaced repetition, get experience
Why Create this Application?

I feel I'm nowhere near living up to my potential as a problem solver and I don't think I'm alone in that feeling. Not performing at our best at solving problems holds us back a great deal at work, and at home.

I also feel really frustrated that I can't remember all of the fundamentals, tips, and tricks I've learned over the years. So I set out to create an online solution that I can use anywhere, anytime.
